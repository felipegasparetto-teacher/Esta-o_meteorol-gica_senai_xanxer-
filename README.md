# Estação Meteorológica - Documentação Completa  Este repositório contém a documentação abrangente de uma estação meteorológica, abrangendo desde a parte eletrônica até a programação. O objetivo é fornecer um guia completo para entusiastas e desenvolvedores que desejam construir e programar sua própria estação meteorológica.
